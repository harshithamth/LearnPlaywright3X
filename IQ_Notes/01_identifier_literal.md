# Identifiers and Literals in JavaScript

| Concept | Meaning | Example |
|---|---|---|
| Identifiers | Names used to identify variables, functions, classes, etc. | `let name;`, `function test(){}` |
| Literals | Fixed values written directly in code. | `42`, `"hello"`, `true` |

| Concept | Purpose | Example |
|---|---|---|
| Identifiers | Used as names | `userName` |
| Literals | Used as values | `"JavaScript"` |
